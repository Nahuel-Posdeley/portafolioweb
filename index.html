Tecnologias

Html: es un lenguaje que permite definir la estructura del contenido de una página web

Bootstrap: es un framework muy utilizado en el desarrollo web. Está basado en librerías CSS y jQuery que nos permiten desarrollar interfaces HTML atractivas y responsive.

CSS: es un lenguaje de programación que se utiliza para definir el estilo y el aspecto de un documento.

Javascript: es el único lenguaje de programación que funciona en los navegadores de forma nativa

GitHub:ayuda a los desarrolladores a almacenar y administrar su código, al igual que llevar un registro y control de cualquier cambio sobre este código.

Visual studio code: es una poderosa herramienta diseñada para desempeñar la tarea específica de escribir y modificar un código web.
